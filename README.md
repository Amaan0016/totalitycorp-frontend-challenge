# E-Commerce App
### Build a Business-to-consumer E-Commerce website that allows people to sell goods and different products.
#### To Start App
`npm i`
`npm run dev`

***
### Tech Stack / Libraries Used :
#### • FrontEnd : <br/>
&ensp;React , Redux , Vite , Material UI<br />
#### • BackEnd : <br />
&ensp;NodeJS <br />
#### • Authentication : <br/>
&ensp;FireBase <br />
